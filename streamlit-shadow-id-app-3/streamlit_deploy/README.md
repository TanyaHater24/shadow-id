# 🔮 Shadow ID — Теневой Паспорт

Нумерологический анализ личности на основе даты рождения.
Powered by Qwen AI.

## Деплой на Streamlit Cloud

1. Создай репозиторий на GitHub
2. Загрузи файлы: `app.py`, `requirements.txt`, `.streamlit/config.toml`
3. Зайди на [share.streamlit.io](https://share.streamlit.io) → New app → выбери репо
4. Deploy!

## Локальный запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```
